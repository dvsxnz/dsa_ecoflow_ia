# EcoFlow IA - Prompt para TCC

Site em ReactJS com Vite criado para apresentar um prompt refinado e uma pipeline de sprints para o TCC EcoFlow, uma cooperativa escolar de reciclagem que transforma resíduos em artesanato sustentável.

## Objetivo da atividade

- Demonstrar uso de técnicas de Inteligência Artificial na criação de prompts.
- Organizar uma pipeline de desenvolvimento em equipe para TCC técnico.
- Entregar um site ReactJS com Vite pronto para publicação no Vercel.
- Disponibilizar o link público do GitHub para avaliação.

## Técnicas de prompt aplicadas

- Persona clara para a IA: gerente de projeto, professor orientador e arquiteto de software.
- Contexto completo do curso técnico, prazo, tecnologias e restrições dos alunos.
- Exemplo guiado com o tema EcoFlow.
- Saída esperada estruturada em backlog, sprints, papéis, riscos, evidências e deploy.
- Restrições explícitas para mobile-first e uso de 13 aulas semanais de 45 minutos.

## Tecnologias

- ReactJS
- Vite
- CSS responsivo mobile-first
- Deploy recomendado: Vercel

## Como executar

```bash
npm install
npm run dev
```

## Como gerar build

```bash
npm run build
```

## Deploy no Vercel

1. Publique o repositório no GitHub.
2. Importe o projeto no Vercel.
3. Use o framework preset `Vite`.
4. Confirme o comando de build `npm run build` e diretório de saída `dist`.
5. Envie o link público do GitHub na tarefa.
