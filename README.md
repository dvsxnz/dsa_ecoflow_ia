# EcoFlow IA - Prompt para TCC

Site em ReactJS com Vite criado para apresentar um prompt refinado e uma pipeline de sprints para o TCC EcoFlow, uma cooperativa de reciclagem e artesanato sustentável.

## Tecnologias

- ReactJS
- Vite
- CSS responsivo mobile-first
- Deploy recomendado: Vercel

## Como executar

```bash
npm install
npm run dev
```

## Como gerar build

```bash
npm run build
```

## Deploy no Vercel

1. Publique o repositório no GitHub.
2. Importe o projeto no Vercel.
3. Use o framework preset `Vite`.
4. Confirme o comando de build `npm run build` e diretório de saída `dist`.
